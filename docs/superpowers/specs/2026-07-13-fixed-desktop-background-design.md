# PC背景固定の設計

## 結論

PC表示の「深い水辺」背景をビューポートに固定し、コンテンツをスクロールしても背景の位置を変えない。現在の斜め4色レイヤー、透明な曲線輪郭、幅480pxの中央コンテンツは維持する。

## 実装

- `.site-stage` の背景へ `background-attachment: fixed` を指定する
- 曲線輪郭は既存の `position: fixed` を維持する
- 480px以下では曲線輪郭を非表示にする既存動作を維持する
- HTML、Reactコンポーネント、Firebase処理は変更しない

## 検証

- CSS回帰テストで背景の固定指定を確認する
- PC幅でスクロール前後の背景位置が同じであることをブラウザで確認する
- スマホ幅で横スクロールやレイアウト崩れがないことを確認する
