# 施設カードのリンク表示強化

## 結論

施設一覧のリンクカード右端に `→` を常設し、カードが詳細ページへ進めることを視覚的に明示する。ホバー時はカードをわずかに上へ移動し、影と枠色を強め、矢印を右へ移動させる。

## 方針

- 既存の `a.facility-card` 構造と詳細ページ遷移を維持する。
- 矢印はカード内の独立した要素として表示し、カード全体のリンク操作を妨げない。
- `:hover` で `transform`、`box-shadow`、`border-color`、矢印の `transform` と色を変化させる。
- 既存の `prefers-reduced-motion: no-preference` のアニメーション制御を維持し、通常時のホバー変化も低減モーション環境では動きを抑える。
- 既存の `:focus-visible` によるキーボード操作の視認性を維持する。

## テスト

- 各施設カードにリンク先を示す `→` が表示されることを確認する。
- 既存のカードクリックで詳細ページへ移動するテストを維持する。
- CSS のホバー状態はクラス構造とスタイル定義を差分確認し、ブラウザでの表示確認時にカードの浮き上がりと矢印移動を確認する。
