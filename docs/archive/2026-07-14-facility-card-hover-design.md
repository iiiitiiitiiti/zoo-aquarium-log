# 施設カードのリンク表示強化

## 結論

施設一覧のリンクカード右端に `→` を常設し、カードが詳細ページへ進めることを視覚的に明示する。ホバー時はカードの背景色だけを淡い緑に変える。

## 方針

- 既存の `a.facility-card` 構造と詳細ページ遷移を維持する。
- 矢印はカード内の独立した要素として表示し、カード全体のリンク操作を妨げない。
- `:hover` で `background` のみを変化させ、カードの位置・影・矢印の位置は変えない。
- 背景色の変化には短いトランジションを使い、低減モーション環境ではトランジションを無効にする。
- 既存の `:focus-visible` によるキーボード操作の視認性を維持する。

## テスト

- 各施設カードにリンク先を示す `→` が表示されることを確認する。
- 既存のカードクリックで詳細ページへ移動するテストを維持する。
- CSS のホバー状態は背景色とトランジションのスタイル定義を差分確認し、ブラウザでの表示確認時に背景色だけが変わることを確認する。
