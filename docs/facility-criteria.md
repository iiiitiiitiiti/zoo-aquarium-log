# 施設マスタの包含基準と ID 規約

## 包含基準

- 常設で一般来場でき、生きた動物・水生生物の展示を主目的とする施設を含める。
- 動物園、水族館、サファリパーク、鳥類園、爬虫類館、昆虫館を含める。
- 牧場・ふれあい施設は、動物展示が主要な来場目的なら含める。
- 植物園・公園・遊園地内でも、独立した常設展示エリアなら含める。
- ペットショップ、動物カフェ、研究施設、期間限定展示、移動動物園・移動水族館は除外する。
- 休園中は `suspended`、閉園確認済みは `closed` として記録を保持する。
- 同一敷地でも別料金・別入口なら別施設、共通入場券なら原則として1施設にまとめる。

## ID 規約

- マスタ施設は `都道府県ローマ字_施設名ローマ字` 形式の小文字スネークケースとする。
- 一度採番した ID は名称変更・移転・閉園後も変更せず、別施設へ再利用しない。
- 手動追加施設は `custom_` と UUID を組み合わせる。
- 重複施設が判明しても自動統合せず、表示上の注意として扱う。
